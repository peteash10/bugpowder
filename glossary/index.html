Glossary of zine terms

(This is an endless work in progress and will probably never be finished. If there are any terms on this site that you don’t understand, please let me know so I can add them here!)

Formats

Like all publications, zines come in a variety of sizes but due to standardisation in reproductive technology there are a few dominant trends.

In the UK and Europe (everywhere outside the USA really) ISO 216 rules the roost. A4 is the most common size of paper to go in a photocopier and it folds perfectly in half to A5, giving us the A5 booklet format, probably the most common way to do a zine.

A3 paper folds to A4 giving us a typical magazine size A4 can be cut and folded down to A6, A7 and smaller.

For reasons best known only to themselves the USA uses their own weird system based around the Letter size and the less said about this nonsense the better. Thankfully Letter is roughly the same size as A4 so we don’t have to worry about it too much.

In summary:

  • A4 : like a normal magazine.
  • A5 : half A4, like a pamphlet.
  • A6 : half A5, fits in your pocket.
  • A7 : half A6, might fit in your wallet.
  • A8 : half A7, credit card size.

A common format in this collection is the US Comic Book size. This does not fit any standard system but is similar in size to a hardback book (or ‘C’ format’ in the book trade). The current standard, as used by the main comic book publishers, is about 6.625 x 10.25 inches.

There are a number of zines in the collection which are this format but which are not comics, and for this reason I wish there was a better term for the size. If you know of one please let me know.

Photocopying

I’ve watched with interest as the photocopier of my youth has been gradually replaced with a printer-scanner combo which makes a digital scan of the document, stores it in memory and prints out a number of copies. This is not how a “real” photocopier works.
